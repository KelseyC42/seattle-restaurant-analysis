# Seattle Restaurant Analysis

## Overview

This project explores restaurant data in downtown Seattle using Yelp and Google reviews to identify patterns in ratings, price levels, and dining experiences. The goal was to uncover hidden gems, with a focus on finding spaces that are more relaxed and suitable for solo or introvert-friendly dining.

## Project Goals

* Analyze restaurant ratings and review patterns
* Identify relationships between price level and customer satisfaction
* Explore sentiment and qualitative feedback from reviews
* Highlight restaurants that provide a quieter, more comfortable atmosphere

## Data Sources

* Yelp reviews
* Google Maps reviews
* Manually curated dataset of Seattle restaurants

## Tools & Skills Used

* R
* RStudio
* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Data visualization
* R Markdown for reporting

## Key Insights

* Higher price does not always correlate with higher satisfaction
* Some lower-profile restaurants consistently receive strong positive feedback
* Atmosphere (quiet vs. loud) plays a meaningful role in user experience
* Review sentiment helps surface “hidden gems” beyond star ratings

## Project Structure

* `restaurant_analysis.Rmd` → Source code and analysis
* `restaurants.xlsx` → Dataset
